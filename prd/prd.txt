

📄 Product Requirements Document (PRD)

프로젝트명: 빠른 위도경도 찾기 웹서비스
PM: ChatGPT
작성일: 2025-06-22
버전: v1.0

⸻

🔍 1. 제품 개요

목표:
누구나 쉽게 주소를 입력하거나 지도에서 이동하여 위도/경도(Lat/Lng)를 빠르게 확인하고 복사할 수 있는 초경량 웹서비스 제공. 모바일 완벽 대응과 함께 SEO 최적화로 “위도 경도 찾기” 검색 시 구글 상위 노출을 목표.

타깃 사용자:
	•	지도 개발을 하는 프론트엔드/백엔드 개발자
	•	부동산/물류/마케팅 등 위치 기반 데이터를 다루는 실무자
	•	일반 사용자 중 위경도 확인이 필요한 유저

⸻

✨ 2. 주요 기능 요약

2.1 주소 입력 → 지도 표시 → 위경도 출력 및 복사
	•	주소 입력 시 네이버 지도 API를 통해 해당 위치 마커 표시
	•	지도 우측 또는 하단에 위도/경도 표시
	•	위도, 경도는 각각 버튼을 통해 클립보드 자동 복사
	•	모바일/데스크탑 모두 동일한 UX 제공

2.2 지도 이동 시 실시간 위경도 표시
	•	사용자가 지도를 이동할 경우 중앙 포인터의 위치 기준으로 실시간 위도/경도 갱신
	•	드래그/핀치 줌 등 제스처 완전 대응

2.3 문의 메일 자동 발송 및 복사
	•	“문의하기” 버튼 클릭 시 사용자의 기본 메일 클라이언트를 열어 mailto: 형식으로 문의 가능
	•	문의 이메일 주소 옆에 클립보드 복사 버튼 제공 (문의: support@example.com [복사])

2.4 검색 이력 저장 및 복원
	•	세션 스토리지 기반으로 사용자가 입력했던 주소 이력 저장
	•	최근 검색 이력 리스트 제공 (최대 5개)

⸻

🧪 3. 기술 스펙

항목	내용
Frontend	Next.js (Static Export)
지도 API	Naver Maps JavaScript API v3 (MCP Context7 적용)
SEO	Next.js의 next/head, Static Meta 태그 최적화, sitemap.xml 및 robots.txt 구성
데이터 저장	sessionStorage
빌드 및 배포	Vercel 또는 정적 호스팅 가능한 서비스 (Netlify 등)
메일 처리	mailto: 프로토콜 + 클립보드 복사 기능
보안 제어	클라이언트 단에서 IP 기반 사용량 제한 (단순 제한) 구현 예정, 추후 백엔드 연동 고려
ui shadcn/ui 기반으로 진행 그리고 component 별로 구분해서 재사용 및 유지 보수 편하게
css 는 tailwind 사용
⸻

📱 4. UI/UX 요구 사항
	•	최대한 심플: 흰 배경 + 검정 텍스트 기반의 미니멀 UI
	•	초기 진입 화면:
	•	“주소를 입력하세요” 인풋 박스
	•	기본 지도 로딩 위치: 대한민국 중심좌표 (예: 서울시청)
	•	반응형 100% 모바일 대응 (최소한 360px width 기준 디자인)
	•	위경도 영역 강조: 크게 보이며, 클릭 시 자동 복사
	•	피드백 UI: “복사되었습니다” 등 최소한의 인터랙션 제공

⸻

🔒 5. 보안 및 남용 방지
	•	클라이언트에서 IP 수집 후 localStorage/sessionStorage 기반 요청 횟수 체크
	•	예: 1분에 20회 이상 좌표 조회 시 alert 또는 제한 메시지 표시
	•	단, 완전한 제한은 백엔드 도입 전까진 우회 가능

⸻

💰 6. 수익화 전략

6.1 Google AdSense 삽입
	•	지도 하단 또는 위경도 복사 영역 아래에 Google AdSense 배너 삽입
	•	정책을 위반하지 않는 선에서 단일 페이지 앱에서도 높은 CTR 유도

6.2 수익 공유 처리
	•	PM 혹은 운영자는 일정 수익(예: 50% 이상)을 가져갈 수 있도록 수익 구조 정의
	•	관련 정책은 내부 협의 후 PRD에 반영 예정

⸻

🎯 7. SEO 및 초기 홍보 전략
	•	제목 태그: 위도 경도 찾기 | 빠르고 쉬운 위치 좌표 확인 서비스
	•	설명 메타 태그: 주소만 입력하면 위도와 경도를 바로 확인하고 복사하세요. 지도 이동도 지원합니다.
	•	OG 태그 및 Twitter Card 지원
	•	Google Search Console 등록
	•	주요 키워드: 위도, 경도, 좌표, 좌표 찾기, 위치 찾기, 위치 좌표, 네이버 지도 좌표
